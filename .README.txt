REQUIREMENTS:
-Ecli(fx)e
This can be obtained by going to the eclipse market place, clicking on the help tab in the top center, clicking on Eclipse Market Place,
and then searching e(fx)clipse. Click 'Install' to install the program.


-JavaFX
If JavaFX isn't installed, the library can be obtained from https://gluonhq.com/products/javafx/




With the JavaFX Library installed and Ecli(fx)e, the program can be run by doing so

1. Create a new project on eclipse

2. Click 'Import Project'

3. Locate the program's file and import it. 

4. to configure javafx, right click on the project and click 'properties'

5. Go to java build path, and click on libraries

6. Click on add Libbrary, click on JRE System, click next

7. Select workspace default JDK 8 / JRE 8, click finish

8. Click on the button 'Add Library"

9. Click on user library, and click next

10. Click on new and name the file as Javafx

11. Click on Add External JARs, locate the javafx library previously downloaded and add all files from the LIB folder

12. Apply and close

13. Run the program

FUNCTIONALITIES:
This program has a total of 4 different options from the main menu
PLAY:
This button enables the user to play the main game
After clicking this button, the user is moved to a setup screen.
Here, the user can create new names to assign player and player 2 to, and can access either standard or chaos mode before starting the game.


In terms of gameplay while using the chessboard, it is similar to normal chess with the exception of no checkmates, however
there are still special moves availible if it is selected in the settings
SETTINGS:
This button enables the user to set settings

Here, the user can modify the color of the white and black player, board, and movement highlighters.

Additionally, they can change certain settings such as modifying the number of consecutive turns one can make in a row,
and changing the number of seconds each player gets to make their turn, however this feature is disabled by default.

Special moves such as castling, promoting pawns, and en passant can be disabled if chosen to by the user.
STATISTICS:
This button enables the user to see some stats regarding the player.
This takes the form of a simple pie chart indicating the win ratio of each opponent, equaling a total of 100%.
CUSTOM CHALLENGE:
Upon clicking this button, the user is taken to a screen containing a menu, containing a single option for now, allowing thme to play
a chess game mode that requires only 1 human's input, a puzzle.
Here, the goal is to complete the puzzle and capture the opposing king in the least amount of turns possible, with a result screen
indicating the least amount of moves needed, and listing how many moves were used trying to complete the puzzle.
